## ‚úÖ SOLUSI: Isi Rencana Aksi Bulanan Terlebih Dahulu

### **LANGKAH 1: Akses Menu SKP Tahunan**

1. Login ke sistem sebagai ASN
2. Klik menu **"SKP Tahunan"** di sidebar
3. Pilih SKP Tahunan untuk **tahun 2026** (atau tahun yang sedang aktif)
4. Pastikan SKP Tahunan sudah **DISETUJUI** oleh Atasan

---

### **LANGKAH 2: Isi Rencana Aksi Bulanan**

1. Di halaman detail SKP Tahunan, cari tombol:
   - **"Isi Rencana Aksi Bulanan"** ATAU
   - **"Edit Rencana Bulanan"** ATAU
   - **"Tambah Rencana Aksi"**

2. Pilih **Bulan** yang akan Anda isi Kinerja Harian
   - Contoh: Jika mau isi Kinerja Harian tanggal **29 Januari 2026**
   - Maka pilih bulan **Januari**

3. Untuk setiap **RHK (Rencana Hasil Kerja)** yang ada, isi:

   **a. Rencana Aksi Bulanan**
   ```
   Contoh: "Evaluasi dan monitoring pelaksanaan kegiatan Januari 2026"
   ```

   **b. Target Bulanan**
   ```
   Contoh: 5
   ```

   **c. Satuan Target**
   ```
   Contoh: "Dokumen" atau "Laporan" atau "File"
   ```

4. Klik **"Simpan"**

5. Status akan berubah dari:
   - ‚ùå `BELUM_DIISI` ‚Üí ‚úÖ `AKTIF`

---

### **LANGKAH 3: Verifikasi di Form Kinerja Harian**

1. Buka menu **"Kinerja Harian"**
2. Klik **"+ Tambah Kinerja Harian"**
3. Pilih tanggal di **bulan yang sudah Anda isi** (contoh: 29 Januari 2026)
4. Dropdown **"Rencana Aksi Bulanan"** sekarang akan muncul dengan pilihan:

   ```
   -- Pilih Rencana Aksi (Opsional) --
   Januari - Evaluasi Kinerja - Evaluasi dan monitoring pelaksanaan...
   Januari - Monitoring Program - Monitoring pelaksanaan program...
   ```

5. Pilih salah satu Rencana Aksi yang sesuai dengan kegiatan Anda hari ini
6. Lanjutkan mengisi form Kinerja Harian

---

## üìÖ TIPS: Isi Rencana Aksi untuk Seluruh Tahun

Agar tidak perlu isi berulang-ulang setiap bulan, **disarankan**:

1. Isi Rencana Aksi Bulanan untuk **12 bulan sekaligus** di awal tahun
2. Untuk setiap bulan (Januari - Desember):
   - Buat rencana aksi yang realistis
   - Sesuaikan target bulanan dengan beban kerja
3. Nanti tinggal pilih saat mengisi Kinerja Harian

**Benefit:**
- ‚úÖ Dropdown selalu ada sepanjang tahun
- ‚úÖ Tidak perlu bolak-balik ke menu SKP Tahunan
- ‚úÖ Lebih efisien

## ‚ùì FAQ (Pertanyaan Umum)

### **Q: Apakah Rencana Aksi Bulanan WAJIB diisi?**
**A:** Tidak wajib, dropdown bertanda **(Opsional)**.
Tapi **SANGAT DISARANKAN** untuk:
- Melacak progress RHK per bulan
- Memudahkan monitoring Atasan
- Otomatis menghitung realisasi target

---

### **Q: Saya sudah isi Rencana Aksi, tapi dropdown masih kosong?**
**A:** Cek:
1. Apakah bulan Rencana Aksi = bulan tanggal yang dipilih di form?
   - Contoh: Rencana Aksi Januari hanya muncul kalau pilih tanggal di Januari
2. Apakah status sudah `AKTIF` (bukan `BELUM_DIISI`)?
3. Refresh halaman (tekan F5)

---

### **Q: Kalau tidak pilih Rencana Aksi, apakah bisa simpan Kinerja Harian?**
**A:** **BISA**. Dropdown opsional.
Tapi kegiatan Anda tidak akan ter-link ke RHK manapun.

---

### **Q: Saya punya 2 RHK, apa harus isi Rencana Aksi untuk keduanya?**
**A:** **Ya**, untuk setiap RHK.
Nanti di dropdown akan muncul 2 pilihan Rencana Aksi (untuk RHK A dan RHK B).
Pilih yang sesuai dengan kegiatan hari itu.

---

### **Q: Target Bulanan itu apa?**
**A:** Jumlah output yang Anda targetkan selesai di bulan tersebut.

**Contoh:**
- RHK: "Evaluasi Kinerja"
- Target Bulanan: 5
- Satuan: "Dokumen"
- Artinya: Target Anda di bulan Januari = menyelesaikan 5 dokumen evaluasi

Nanti saat isi Kinerja Harian, **Progres** akan terakumulasi ke Target Bulanan ini.
